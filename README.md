# React Parcel Starter Kit

React with Parcel bundler starter kit

## Running & Building

- run local: `yarn start`
- run test: `yarn test`
- build bundle: `yarn build`

## Current versions

src/

- React 16.8.x
- Parcel 1.12.x
- ES6

test/

- Jest 24.x
- Babel 7.x (env/react presets)
- Enzyme 3.9.x
