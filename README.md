# feast

Recipe aggregator app. You can find it deployed on Netlify @ https://bit.ly/feast-web

## Web (React / Typescript with PurifyTS)

### Running & Building

- run local: `yarn start` -> port `1234`
- run test: `yarn test`
- build bundle: `yarn build` -> build artifacts in `dist/` folder
